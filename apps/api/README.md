## Installation

- install dependencies
```bash
$ npm install

```
- rename docker.env-template with docker.env and set configuration data

## Running the app


- watch mode in docker
```bash
$ npm run start:docker
```
- //TODO: update this] production mode
```bash
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
